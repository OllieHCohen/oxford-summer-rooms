# Oxford Summer Rooms

Static site for **OxfordSummerRooms.com** — short-term Oxford student rooms available for summer lets.

Single static page (`index.html`, no build step). Live property listings are read client-side from the Supabase backend, gated on `property_live_status.is_live`. Deployed on Vercel.

Rebranded from the original "Regalis Property" holding page: new logo, hero text removed, listings/functionality unchanged.
